# Analisis_ventas_excel 🚀

Script de automatización en Python desarrollado con la librería `openpyxl` para la generación automática de reportes de ventas en Excel. El programa crea un libro de trabajo desde cero, estructura encabezados estandarizados y procesa dinámicamente un conjunto de datos comerciales de hardware. Automatiza el cálculo de métricas financieras mediante la inserción de fórmulas nativas de Excel (MULTIPLICACIÓN y SUMA) para calcular totales y el gran total general. Finalmente, aplica estilos profesionales personalizados (fuente Arial, negritas y rellenos de color sólido) para mejorar la presentación del documento final de forma eficiente.

## 🛠️ Características del Proyecto
* **Generación dinámica:** Estructura datos sin importar la cantidad de registros.
* **Fórmulas nativas:** Inserción de funciones matemáticas automáticas directamente en las celdas.
* **Diseño profesional:** Aplicación de estilos visuales, fuentes y colores corporativos de manera programática.

## 📦 Requisitos
Para ejecutar este script, necesitas instalar la librería:
```bash
pip install openpyxl
```
